# Quiz Show Game

เกมตอบคำถามสดแบบ Kahoot-style สำหรับใช้ในงานอีเวนต์ ห้องเรียน หรือกิจกรรมชิงรางวัล

## วิธีเปิดเกม

ดับเบิลคลิก `start-game.cmd` หรือเปิดคำสั่งนี้ในโฟลเดอร์โปรเจกต์:

```powershell
node server.js
```

จากนั้นเปิด:

- Admin: http://localhost:3000/admin
- จอใหญ่: http://localhost:3000/screen
- มือถือผู้เล่น: สแกน QR จากหน้า Admin หรือจอใหญ่

ถ้าให้ผู้เล่นใช้มือถือใน Wi-Fi เดียวกัน ให้ใช้ลิงก์ Network ที่เซิร์ฟเวอร์แสดง เช่น `http://192.168.x.x:3000/admin`

## ฟีเจอร์หลัก

- สร้างชุดคำถามได้หลายชุด
- คำถามรองรับข้อความและรูปภาพ
- คำถามละ 4 ตัวเลือก และกำหนดคำตอบที่ถูก
- เปิดห้องด้วย PIN 6 หลักและ QR Code
- ผู้เล่นใส่ชื่อเล่นผ่านมือถือ
- แอดมินกด Start, เฉลย, ข้อต่อไป และจบเกม
- เวลาข้อละ 20 วินาที
- คะแนนคิดจากตอบถูกและความเร็ว
- จบเกมแสดงผู้ชนะ 5 อันดับบนจอใหญ่
- เก็บรายชื่อผู้เล่นในห้องระหว่างเกม
- เพลงและเสียงประกอบสร้างจาก Web Audio ในเบราว์เซอร์

## หมายเหตุสำหรับงานจริง

ระบบนี้ออกแบบให้รองรับผู้เล่นอย่างน้อย 200 คนบนเครือข่ายเดียวกัน โดยใช้ Server-Sent Events และ HTTP API เพื่อลดความซับซ้อนของการติดตั้ง
